# Prospecting Email — 公司背调 + 开发信生成工具

> 独立工具。拖入客户表 → AI 背调 → 输出定制开发信。

---

## 使用方法

1. 把客户表（Excel/CSV）拖入当前文件夹
2. 对 Claude 说：**"分析客户表"** 或 **"生成开发信"**
3. AI 逐家背调 → 输出报告 + 双语开发信

飞书多维表格：提供 Base Token + Table ID 即可直接读取。

---

## 输入格式要求

表格至少包含「公司名」列。额外的网站、国家、品类列可大幅提升背调精度。

---

## 输出

- 每家公司的背调信息卡（官网/规模/品类/进口特征/决策人/近期动态）
- 客制化双语开发信（西语/葡语 + 英语）
- 报告保存至 `reports/客户背调报告-YYYY-MM-DD.md`

---

## 目标市场

默认拉美（墨西哥、巴西、智利、秘鲁、哥伦比亚、阿根廷），西语/葡语为主。
如需其他市场，直接说明即可。

---

## 可用工具

- **WebSearch**：公司信息 + 新闻搜索
- **WebFetch**：官网深度抓取
- **tavily-search**：新闻背调（`tvly search --topic news`）
- **lark-cli**：飞书多维表格读写（如客户表在飞书）
