## 三、广告词 & 垃圾词黑名单

### 2.1 硬禁止（触发即不合格）

| 类别 | 西语禁止词 | 英语禁止词 | 原因 |
|------|-----------|-----------|------|
| 最高级 | más grande, el mejor, número uno, insuperable, único, incomparable, excepcional, extraordinario | best, largest, #1, number one, unmatched, unbeatable, exceptional, extraordinary | 垃圾过滤器首要打击对象 |
| 制造紧迫 | urgente, actúa ahora, no esperes, última oportunidad, tiempo limitado, date prisa, corre | urgent, act now, don't wait, last chance, limited time, hurry, run | 紧迫感 = 垃圾信号 |
| 夸大承诺 | garantizado, 100% seguro, sin riesgo, resultados asegurados, cero preocupaciones | guaranteed, 100%, risk-free, results guaranteed, zero worries | 过度承诺触发过滤 |
| 价格诱饵 | oferta especial, descuento exclusivo, precio increíble, ganga, barato, rebaja, promoción | special offer, exclusive discount, amazing price, deal, cheap, sale, promotion | 促销词 = 广告/垃圾 |
| 排名宣称 | top 5, top 10, los primeros, entre los mejores, clasificado, rankeado | top 5, top 10, ranked, rated #, leading the pack | 无第三方验证的排名 = 虚假广告 |
| 夸张金额 | millones de dólares, fortunas, miles de millones, enriquecer | millions of dollars, fortune, billions, get rich | 对拉美客户，用实体资产替代估值 |
| 全大写 | CUALQUIER PALABRA EN MAYÚSCULAS (含主题行) | ANY ALL-CAPS WORD (including subject lines) | 垃圾邮件典型特征 |
| 感叹号 | ! | ! | 全文不用，包括主题行 |

### 2.2 软限制（可用但需控制）

| 词/表达 | 限制 |
|---------|------|
| líder / leading | 每封 ≤1 次，需有事实支撑 |
| sin costo / at no cost | 每封 ≤1 次 |
| gratis / free | 每封 0 次 |
| competitivo / competitive | 每封 0 次 |
| eficiente / efficient | 每封 0 次 |

### 2.3 特别规则

- ❌ 不出现任何排名性表述
- ❌ 同一封邮件不出现船东名 + 具体运价数字
- ❌ 不对海外客户使用 digital / AI / 平台 / technology 等技术词汇
- ❌ 不使用「!!!」「…」「???」等多重标点
- ❌ 不在首段出现「somos」「we are」等自我介绍（Hook 段落说客户，不说自己）

## 四、模板使用规范检查清单

发送前逐项确认：

- [ ] 对象类型正确？（代理 / 直客 / 未标签）
- [ ] 广告垃圾词检查：无最高级/紧迫词/夸大承诺/价格诱饵/排名宣称/全大写/感叹号？
- [ ] 代理模板未提及本地仓库、本地团队？
- [ ] 直客 Proof 选用了本地化相关变体？
- [ ] 同一封未同时出现船东名 + 具体运价数字？
- [ ] 未使用 digital / AI / 平台 / technology 等技术词汇？
- [ ] 主题行与同序列前几封完全一致？
- [ ] 西语在前、英语在后？
- [ ] 全文第二人称，没有教客户做事的口吻？
- [ ] 首段没有出现自我介绍（Hook 说客户，不说自己）？
- [ ] CTA 是「给」不是「要」？
- [ ] 占位符全部替换完毕？
- [ ] 同公司序列内模板已轮换？
