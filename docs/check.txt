模板使用规范检查清单

写信前后逐项确认：

- [ ] 对象类型正确？（代理 / 直客 / 未标签）
- [ ] **广告垃圾词检查：无最高级/紧迫词/夸大承诺/价格诱饵/排名宣称/全大写/感叹号？**
- [ ] **软限制：líder 不超过 1 次且有事実支撑？无语空洞形容词（competitivo/eficiente）？**
- [ ] 代理模板未提及本地仓库、本地团队？
- [ ] 同一封未同时出现船东名 + 具体运价数字？
- [ ] 未使用 digital / AI / 平台 / technology 等技术词汇（对海外客户）？
- [ ] 全文第二人称，没有教客户做事的口吻？
- [ ] 首段没有出现「Somos...」「We are...」（Hook 说客户，不说自己）？
- [ ] CTA 是「给」不是「要」？
- [ ] 占位符全部替换完毕，无残留 `[XXX]`？
- [ ] 是否都是实时数据
