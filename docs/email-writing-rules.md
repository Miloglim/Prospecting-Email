# 开发信写作准则（拉美 B2B 冷邮件）

> 每封开发信必须遵守。违反任何一条硬约束 = 不合格。

## 黄金公式

```
Hook（钩子） + Pain Point（痛点共鸣） + Proof（证据） + CTA（微小行动）
```

| 要素 | 作用 | 正确写法 | 错误写法 |
|------|------|---------|---------|
| **Hook** | 3 秒停下滑动 | "Entiendo que importan desde..."（我了解到…） | "Ustedes importan 293 embarques…"（你在教客户做事） |
| **Pain Point** | 让客户觉得"你懂我" | 从客户的视角描述困境——"cuando X falla, tu línea se para" | 从行业角度泛泛而谈 |
| **Proof** | 用具体数字建立信任 | "帮过同品类客户，清关通过率 95%+" | 纯自我推销 |
| **CTA** | 给客户东西 | "Te preparo X, sin costo" | "需要报价吗？" |

### 视角铁律

1. **不做客户的老师**：不要写"你们公司做XX行业，每年进口YY"。客户比你更了解自己的业务。
2. **用「我了解到」代替「你们是」**：Hook 用 "Entiendo que…" "Noté que…" "Vi que…"，展现你做了功课，但不在教客户。
3. **第二人称 + 为客户着想**：每句话问自己——「这句话是在展示我多厉害，还是在帮客户解决问题？」
4. **尊重客户的现有安排**：不要说「你的货代不行」，要说「多一个备选，多一份安心」。

## 九条硬约束（违反 = 重写）

| # | 规则 | 错误 → 正确 |
|---|------|-----------|
| 1 | 🎯 **标题有信息量** | ❌ "Hello""合作""报价" → ✅ "[痛点]+[数字]，你受影响了吗？" |
| 2 | 👤 **第一句说客户，不说自己** | ❌ "我们是 XX 公司…" → ✅ "看贵司主要做 XX 出口巴西" |
| 3 | 📏 **150 词内**（西语）/ **120 词内**（葡语） | ❌ 长篇大论 → ✅ 一句一行，30 秒读完 |
| 4 | 🎯 **一封只讲一个价值** | ❌ "海运空运海外仓报关我们都能做" → ✅ "墨西哥正清通关 3-5 天" |
| 5 | 📊 **用具体数字，不用模糊词** | ❌ "价格好""服务优""效率高" → ✅ "清关通过率 95%+""300+ 船司" |
| 6 | 🌐 **西语/葡语开头** | ❌ 纯英文 → ✅ 西语/葡语在前 |
| 7 | 🎁 **CTA 是"给"不是"要"** | ❌ "需要报价吗？" → ✅ "发一份 XX 分析给你参考？免费的" |
| 8 | 📱 **签名档必带 WhatsApp** | ❌ 只有邮箱 → ✅ WhatsApp: +86 XXX |
| 9 | 🔗 **不带大附件** | ❌ 5MB PDF → ✅ "需详细资料可回复或点链接" |

## 三条铁律（踩了必死）

| 🚫 铁律 | 为什么致命 |
|---------|----------|
| **不拼低价** | 拉美线真正值钱的是合规、稳定、清关能力——拼低价=放弃溢价 |
| **不卖灰色/不合规服务** | 合规是唯一长期出路——卖合规短期获客慢，但客户质量高、不暴雷 |
| **不群发风格** | 每封必须让客户觉得是专门写给 ta 的 |

## 跟进节奏

| 热度 | 频率 | 原则 |
|:--:|------|------|
| 🔥 热（已报价） | 2-3 天 | 快速响应 |
| 🌡️ 温（已回复） | 5-7 天 | 每次带新情报 |
| ❄️ 冷（未回复） | 7-14 天 | 不催促，只给价值 |
| 🧊 冰（3次+无回复） | 1-2 月 | 只发高价值内容 |

- 跟进必须带新信息（新关税/新港口动态/新案例），不能只是 "checking in"
- 跟进用纯西语/葡语
- 第 5 次跟进无回复 → 降级为月度行业报告订阅

## 开发信结构模板

```
Asunto: [具体痛点+数字钩子，3-12词，来自背调发现]

[Hook — 2-3句]
→ 基于背调的具体发现破冰。不说"我们是XX公司"。
→ 自然口语化，像 WhatsApp 消息。

[Pain Point — 2-3句]
→ 行业痛点共鸣，用背调中发现的数字。
→ 引用与客户业务相关的具体数据。

[Proof — 2句]
→ YQN 是谁，能帮什么，带 company-data.md 中的具体数据。
→ 低调但有份量。只讲和这个客户相关的点。

[CTA — 1句]
→ 给东西不要东西。微小、零压力、可忽略。
→ 四种类型轮换使用，避免千篇一律：

| 类型 | 示例 | 适用场景 |
|------|------|---------|
| 📄 **发资料** | "Te preparo un análisis sin costo" | 客户有明确物流体量，数据能说服 |
| ☕ **轻聊** | "Si te parece, podemos charlar 5 minutos" | 客户体量大、值得建立关系 |
| 🎁 **分享洞察** | "¿Te sirve que te comparta un comparativo?" | 信息不足时，先给后问 |
| 🤝 **留门** | "Sin prisa, quedo a tus órdenes" | 跟进多次后，把主动权完全交给客户 |

> 🚫 禁止：预约型（"agendemos una llamada"）、索取型（"contáctame"）、催促型（"¿cuándo podemos hablar?"）

[签名档]
金颖哲 Zayne Jin | Overseas Sales · LatAm Desk
YQN Logistics Technology Group
📧 zayne_jin@trimanshipping.com | 📱 +86 18487665870 | 🌐 www.yqn.com
```

## 西语质量检查清单

- [ ] 不是机器翻译腔——读起来像人说的话
- [ ] 第一段不超过 50 词（3 秒能读完）
- [ ] 没有 "Somos..." 开头
- [ ] 没有催促——"cuando puedas""sin prisa"
- [ ] 全文 < 150 词
- [ ] 用词符合目标国家习惯（墨西哥哥用 celular 不是 móvil）
- [ ] 数字加粗、CTA 加粗、差异化卖点加粗（每段 2-4 处，全文 ≤ 8 处）
- [ ] 人称一致（usted 全文统一，不混用 tú/usted）
